# USB-Powered ATmega328P LED Controller PCB

## Overview
This project is a custom-designed USB-powered LED controller PCB based on the ATmega328P microcontroller. The board is designed for simple LED control applications and demonstrates complete PCB design workflow using KiCad.

## Features
- ATmega328P microcontroller
- USB Micro-B power input
- On-board LEDs with current limiting
- Compact, single-layer PCB
- DRC-clean and fabrication-ready design

## Tools Used
- KiCad (Schematic & PCB Design)
- ATmega328P
- USB Micro-B Connector
